# TaylorSeries
"I have developed a Python program that simplifies complex mathematical functions by calculating their Taylor series expansions centered at x = 0 (Maclaurin series). 
The code currently supports the expansion and simplification of functions like sin(x) and e^x. 
As a personal challenge, I chose to implement my own custom versions of the built-in pow() function and the factorial function (traditionally from the math library). 
The simplification process runs iteratively, continuing until the absolute error drops below a predefined threshold.
